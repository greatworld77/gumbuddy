🛠️ Quick Setup Guide

Step 1: Create Telegram Bot

Message @BotFather on Telegram and use /newbot to create your bot. Save the API token.

Step 2: Get Novita AI Key

Sign up at Novita AI and generate an API key from your dashboard.

Step 3: Deploy to Vercel

Click the button below to deploy instantly:

Deploy with Vercel
Step 4: Set Environment Variables

Add these to your Vercel project:

TELEGRAM_TOKEN=your_telegram_bot_token
NOVITA_API_KEY=your_novita_api_key
Step 5: Set Webhook

Visit this URL in your browser (replace with your details):

https://api.telegram.org/bot<YOUR_BOT_TOKEN>/setWebhook?url=https://<YOUR_VERCEL_URL>/api/webhook
